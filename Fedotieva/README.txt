Программа скачивает с сайта https://sne.space данные о сверхновой, название которой вводится в качестве аргумента командной строки, и выводит значение максимальной звездной величины, соответствующий момент времени и информацию об источнике данных. Данные выводятся для всех фотометрических полос, представленных на сайте для данной сверхновой.

Замечание: Программа ищет максимум среди имеющихся точек кривой блеска, поэтому если в данном фильтре сверхновая наблюдалась после пикового значения звездной величины, то получаемое значение не будет соответствовать максимуму блеска.
